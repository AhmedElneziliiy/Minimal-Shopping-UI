# Minimal Shopping UI — React App

A **minimal React shopping interface** for browsing and viewing products — built with React Router for navigation and a clean component-based architecture.

## What it does

A lightweight e-commerce frontend that displays a product list, individual product detail pages, and a home page. Products are fetched and rendered through reusable React components.

## Tech Stack

- **React** — functional components with hooks
- **React Router** — client-side routing
- **CSS** — component-scoped styles

## Key Components

| Component | Description |
|---|---|
| `Home` | Landing/home page |
| `Products` | Product grid listing |
| `ProductCard` | Single product card |
| `SingleProduct` | Product detail page |
| `NavBar` | Site-wide navigation |
| `NoMatch` | 404 fallback page |

## Getting Started

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
